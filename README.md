# CARB-APPLE-V3.0

NOW FOR APPLE

## About

This is an Expo-based mobile application for iOS.

## Getting Started

### Prerequisites

- Node.js installed
- Expo Go app on your iOS device (for testing)
- Xcode (for iOS development)

### Installation

```bash
npm install
```

### Running the App

```bash
# Start the development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android

# Run on web
npm run web
```

## Project Structure

- `/app` - Application screens and navigation (using Expo Router)
- `/assets` - Images, fonts, and other static assets
- `app.json` - Expo configuration
- `package.json` - Project dependencies

## Expo SDK

This project uses Expo SDK 52.
